# PL_Search
根据ArticleSpider（https://github.com/xiaorui930822/ArticleSpider）爬取的数据使用Django搭建搜索引擎

本项目在Linux下配合nginx+uwsgi发布到服务器上，相关参数设置可参照文件中conf文件和ini文件；

本项目中使用django-pure-pagination对页码进行了处理，可用但是仍缺css样式，后期补充；
